# Information
エンターキーを押さなくてもキー入力を拾い上げ定期的にpublishするノード．文字入力がない場合は'\0'文字をpublishする．
例えばOpenAI Gymなどで，'0'，'1'と押すだけで行動入力してくれる('0'+エンターキー，'1'+エンターキーとしなくてよい)．
